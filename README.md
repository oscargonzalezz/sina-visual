# SINA-VISUAL

SINA-VISUAL es una aplicación en C que implementa y visualiza una red neuronal multicapa interactiva utilizando **Raylib**.

## Estructura del Proyecto

*   `src/`: Código fuente (`main.c`, `app.c`, `sinacore.c`, `sinacore.h`).
*   `data/`: Archivos de configuración (`config.txt`) y datasets.
*   `sina_visual.dev`: Archivo de proyecto para Dev-C++.

## Requisitos

*   Compilador C (GCC/MinGW recomendado).
*   **Raylib** (OpenGL32, GDI32, WinMM, ComDlg32).

## Compilación y Ejecución

### Desde Dev-C++
Abre el archivo `sina_visual.dev` y presiona **F11** (Compilar y Ejecutar).

### Desde la Consola (Windows)
Ejecuta el script incluido:
```bash
compilar_y_ejecutar.bat
```
